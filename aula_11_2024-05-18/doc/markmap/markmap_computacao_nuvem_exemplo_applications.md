# Aplicações

## Modelos de Serviço

* **SaaS (Software como Serviço)** 
  * Google Docs 
  * Microsoft Office 365 
  * Dropbox
* **PaaS (Plataforma como Serviço)** 
  * AWS Elastic Beanstalk 
  * GCP App Engine 
  * Azure App Service
* **IaaS (Infraestrutura como Serviço)**  
  * AWS EC2 
  * GCP Compute Engine 
  * Azure Virtual Machines
* **FaaS (Função como Serviço)**  
  * AWS Lambda 
  * GCP Functions 
  * Azure Functions
